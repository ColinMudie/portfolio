Planet Fitness Quiz
Github link

I am currently a shift at my local Planet Fitness in Mt. Pleasant, MI. I decided to create this quiz as a fun way for my staff to test their knowledge of our club or planet fitness as a whole. I had created a timed test in the past and wasn't happy with the results so I wanted to spend the time to understand what I didn't like about my last version. With this one, I used an interval timer that decreases with each incorrect answer. I also used a second timer to briefly display whether the selected answer was correct or incorrect by changing the background color in the text area.

Find It Here
Github link

This is a basic movie search program that attempts to solve the issue of looking up a movie on a Netflix, only to find it is only available on Hulu. This Project was made with the help of James Gault and Javier Sevilla. This website takes the users search input and through the use of web APIs, it will return the location you can find the movie at, data from imdb, as well as a recommended and previously viewed list. Since we were a team I was focused on delegating tasks for my team members as well as gathering and pinning the API data with AJAX calls in JS to the HTML.
